This project builds a machine learning model to predict whether a credit card applicant is a good or bad client using application data and credit history.
The project follows an end-to-end data science workflow including data cleaning, exploratory data analysis, and modeling.

Data Source : https://www.kaggle.com/datasets/rikdifos/credit-card-approval-prediction

Due to GitHub file size limits, the dataset is not included in this repository.
You can download the dataset from Kaggle

## How to Run
1. Download the dataset from Kaggle
2. Place the CSV files in a the same folder
3. Run the notebooks in order:
   - credit-card-clean.ipynb
   - credit-card-visualization.ipynb
   - credit-card-modeling.ipynb
