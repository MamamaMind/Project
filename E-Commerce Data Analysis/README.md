## Overview
This project analyzes an end-to-end e-commerce sales pipeline using a star schema dataset (1 fact table + 5 dimension tables). The goal is to understand sales performance, customer behavior, product trends, payment patterns, store performance, and time-based trends through data visualization and business insights.

## Dataset
Data Source : https://www.kaggle.com/datasets/mmohaiminulislam/ecommerce-data-analysis

Schema: Fact table (transactions) linked to Customer, Item, Store, Time, and Payment dimension tables.
